# WhatsApp Ordering Website

A simple WhatsApp-based ordering website built with **HTML, CSS, and JavaScript**.  
This project allows customers to select products, enter their personal details, and send their order receipt directly to the business owner’s WhatsApp without using online payments or a backend server.

---

## 🚀 Features

- Display products with prices
- Add items to a shopping cart
- Collect customer details (name, phone number, address)
- Automatically generate an order summary (receipt)
- Send the order directly to the owner’s WhatsApp using click-to-chat
- No backend, no database, no payment gateway
- Runs locally in the browser

---

## 🛠️ Technologies Used

- HTML5 – page structure
- CSS3 – basic styling
- JavaScript (Vanilla) – cart logic and WhatsApp integration
